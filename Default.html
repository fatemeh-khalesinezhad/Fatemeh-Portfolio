    My Learning Path

    Who Am I?

    Hello! I'm Nazanin, a passionate web developer and space enthusiast. My journey into the world of coding began with curiosity and has evolved into a deep love for creating beautiful, interactive web experiences. This portfolio represents not just my technical skills, but also my creative vision and dedication to learning.

    How I Started

    My coding journey began with simple HTML pages, gradually expanding into CSS styling and JavaScript interactivity. I was fascinated by how lines of code could transform into beautiful, functional websites. The combination of logic and creativity in web development captured my imagination and motivated me to dive deeper into this field.

    Who Helped Me

    I am deeply grateful to my teacher, Dr. Majid Tajan-Jari. A brilliance that seemed to have come from the future; whose guidance and encouragement have been instrumental in my growth as a developer and AI devotee. His company's teaching style, which balances technical rigor with creative freedom, has inspired me to push boundaries and create something truly unique. I also owe thanks to my online friends whose resources and support have been invaluable.

    Building This Website

    This cosmic portfolio was built from scratch using modern web technologies and AI assistance. The process involved careful planning, iterative design, and countless hours of coding and refinement. I wanted to create something that wasn't just functional, but also visually stunning and engaging. Every animation, every interaction was crafted with attention to detail and user experience in mind.

    My Tools & Technologies

    Core Technologies: HTML5, CSS3, JavaScript
    Animation Library: GSAP (GreenSock Animation Platform)
    Design Tools: Custom CSS animations, SVG graphics, AI assistance
    Development Environment: Modern code editor with live preview
    Techniques: Responsive design, glassmorphism, parallax effects, 3D transforms, professional AI prompting
